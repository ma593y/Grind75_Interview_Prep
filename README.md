## How to Use This Repo

### File Types
Each problem has **two** markdown files:
- **`*-solution.md`** (Full Solution): Complete problem breakdown with problem statement, concept overview, brute force + optimal approaches with Python code, complexity analysis, edge cases, and related problems.
- **`*-study.md`** (Study Guide): Condensed review-focused document with key intuition, multiple approaches with trade-offs, common mistakes, interview tips, and self-assessment checklist.

### Recommended Workflow
1. **First pass**: Read the `-solution.md` file to understand the problem deeply
2. **Practice**: Attempt to solve without looking at code
3. **Review**: Use the `-study.md` file for spaced repetition review
4. **Track**: Update the status column (⬜ -> ✅) as you complete problems

### Suggested Study Order
See [STUDY_GUIDE.md](./STUDY_GUIDE.md) for a recommended study order that builds concepts progressively.
